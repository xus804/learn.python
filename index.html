Lesson 2: Variables üì¶

Variables are like labeled boxes where you store information. You can name them almost anything!

name = "Alice"
age = 25
print(name)
print(age)

Here, name stores text and age stores a number. The = sign assigns values to variables.

Variable naming rules:

  • Can contain letters, numbers, and underscores
  • Must start with a letter or underscore
  • Case sensitive: age and Age are different
  • Can't use reserved words like print, if, etc.

Lesson 5: Strings üìù

Strings are sequences of characters (text). You can do cool things with them!

message = "Hello"
name = "Alice"

# Concatenation (joining)
greeting = message + " " + name
print(greeting) # "Hello Alice"

# String methods
print(message.upper()) # "HELLO"
print(message.lower()) # "hello"
print(len(message)) # 5 (length)

String formatting (f-strings):

name = "Bob"
age = 30
print(f"My name is {name} and I'm {age}")

Lesson 6: Lists üìã

Lists store multiple items in a single variable. They're like shopping lists!

fruits = ["apple", "banana", "cherry"]

print(fruits[0]) # "apple" (first item)
print(fruits[-1]) # "cherry" (last item)
print(len(fruits)) # 3 (number of items)

# Adding items
fruits.append("orange")

# Removing items
fruits.remove("banana")
üí° Note: Python uses 0-based indexing. The first item is at index 0, not 1!

Lesson 7: Conditionals üîÄ

Make decisions in your code using if, elif, and else statements!

age = 18

if age >= 18:
print("You can vote!")
elif age >= 16:
print("Almost there!")
else:
print("Too young to vote")

Comparison operators:

  • == equal to
  • != not equal to
  • > greater than
  • < less than
  • >= greater than or equal
  • <= less than or equal

Lesson 8: Loops üîÑ

Loops let you repeat code multiple times without writing it over and over!

For loops:

# Loop through a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
print(fruit)

# Loop through a range
for i in range(5):
print(i) # Prints 0, 1, 2, 3, 4

While loops:

count = 0
while count < 3:
print(count)
count += 1 # Same as count = count + 1

Lesson 9: Functions üéØ

Functions are reusable blocks of code. Think of them as recipes you can use over and over!

def greet(name):
return f"Hello, {name}!"

message = greet("Alice")
print(message) # "Hello, Alice!"

# Function with multiple parameters
def add(a, b):
return a + b

result = add(5, 3)
print(result) # 8
üí° Key points: Functions start with def, take parameters (inputs), and can return values (outputs).

Lesson 10: Dictionaries üóÇÔ∏è

Dictionaries store data in key-value pairs, like a real dictionary with words and definitions!

person = {
"name": "Alice",
"age": 25,
"city": "New York"
}

print(person["name"]) # "Alice"
print(person["age"]) # 25

# Adding/updating
person["job"] = "Engineer"

# Getting all keys
print(person.keys())

# Getting all values
print(person.values())
